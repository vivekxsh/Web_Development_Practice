## 1. Document structure tags

| Tag       | Description                     |
| --------- | ------------------------------- |
| `<html>`  | Root element of an HTML page    |
| `<head>`  | Contains metadata (not visible) |
| `<title>` | Title shown in browser tab      |
| `<body>`  | Visible content of the webpage  |

---

## 2. Metadata tags (inside `<head>`)

| Tag       | Description                             |
| --------- | --------------------------------------- |
| `<meta>`  | Metadata like charset, author, viewport |
| `<link>`  | Links external files (CSS, icons)       |
| `<style>` | Internal CSS styles                     |
| `<base>`  | Sets base URL for relative links        |
